# 🚀 Todo Pro App

A modern full-stack Todo Application built with **MERN Stack (MongoDB, Express, React, Node.js)** featuring a beautiful dark UI, smooth animations, and advanced task management features.

---

## 🌟 Features

* ✅ Add Tasks  
* 🗑️ Delete Tasks  
* ✏️ Edit Tasks  
* ✔️ Mark Complete / Incomplete  
* 🔍 Search Tasks  
* 🎯 Priority Levels (Low, Medium, High)  
* 🌙 Dark Mode UI  
* 📊 Live Task Counter  
* ✨ Smooth Animations  
* 📱 Responsive Design  
* ☁️ MongoDB Database Integration  

---

## 🛠 Tech Stack

* Frontend: React.js  
* Backend: Node.js + Express.js  
* Database: MongoDB Atlas  
* Styling: CSS (Custom UI + Glass Effect)

## ⚙️ Setup Instructions

### 1️⃣ Clone Repository
git clone https://github.com/Rahul-ghanchi/todo-app.git

cd todo-app

---

### 2️⃣ Backend Setup
cd backend
npm install

Create `.env` file:
MONGO_URI=your_mongodb_connection_string
PORT=5000

Run backend:
node server.js

---

### 3️⃣ Frontend Setup
cd frontend
npm install
npm start

## 🌐 API Endpoints

| Method | Endpoint       | Description   |
| ------ | -------------- | ------------- |
| GET    | /api/tasks     | Get all tasks |
| POST   | /api/tasks     | Add task      |
| PUT    | /api/tasks/:id | Update task   |
| DELETE | /api/tasks/:id | Delete task   |
